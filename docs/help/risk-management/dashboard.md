# Risk Dashboard

The Risk Dashboard provides a comprehensive overview of your organization's risk posture, enabling informed decision-making and effective risk management.

## Overview

The dashboard displays:
- Risk metrics and KPIs
- Risk distribution by category and level
- Trend analysis over time
- Open vs. closed risks
- Treatment progress

## Key Metrics

### Total Risks
- Active risks in the register
- Breakdown by status
- New risks this period

### Risk by Level
Shows distribution across severity levels:
- **Very High / Critical**: Immediate attention required
- **High**: Significant concern
- **Medium**: Moderate attention needed
- **Low**: Minor concern
- **Very Low**: Minimal risk

### Average Risk Score
- Organization-wide average
- Trend indicator (improving/worsening)
- Comparison to previous period

### Treatment Progress
- Risks in treatment
- Completion percentage
- Overdue treatments

## Dashboard Widgets

### Risk Level Distribution
Donut chart showing:
- Count of risks per level
- Percentage breakdown
- Click to filter risk register

### Category Breakdown
Bar chart showing risks by category:
- Operational
- Strategic
- Compliance
- Security
- Financial
- Technical
- Third-Party

### Trend Over Time
Line chart showing:
- Total risks over time
- New risks added
- Risks closed
- Average risk level trend

### Top Risks
Table of highest-priority risks:
- Risk title and ID
- Current risk level
- Owner
- Status
- Quick actions

### Heatmap Preview
Visual risk heatmap:
- Likelihood vs. Impact grid
- Risk count per cell
- Click to open full heatmap

### Recent Activity
Timeline of recent risk actions:
- New risks created
- Assessments completed
- Treatments updated
- Risks closed

## Filters

Filter dashboard data by:

### Time Period
- Last 30 days
- Last quarter
- Last year
- Custom range

### Category
Select specific risk categories

### Owner
Filter by risk owner

### Workspace
If multi-workspace enabled, filter by workspace

## Customization

### Widget Layout
1. Click **Customize**
2. Drag widgets to reorder
3. Resize widgets
4. Hide/show specific widgets
5. Save layout

### Saved Views
Create multiple dashboard views:
1. Configure filters and layout
2. Click **Save View**
3. Name the view
4. Switch between views

## Drill-Down

Click any metric or chart element to:
- View underlying data
- Navigate to filtered risk register
- See detailed breakdown

### Example
Click "High" in the risk level chart to:
- See all high-risk items
- Review details
- Take action

## Export and Share

### Export Dashboard
1. Click **Export**
2. Choose format:
   - PDF (visual report)
   - Excel (data)
   - PowerPoint (presentation)
3. Download or email

### Scheduled Reports
Set up automatic dashboard exports:
1. Go to **Tools → Scheduled Reports**
2. Create new report
3. Select Risk Dashboard
4. Configure schedule
5. Add recipients

### Share Link
Create shareable dashboard link:
1. Click **Share**
2. Configure permissions
3. Copy link
4. Recipients can view (read-only)

## Risk KPIs

### Key Performance Indicators

| KPI | Description | Target |
|-----|-------------|--------|
| Average Time to Assess | Days from creation to assessment | < 14 days |
| Average Time to Treat | Days from assessment to treatment start | < 30 days |
| Overdue Assessments | Risks waiting for assessment | 0 |
| Treatment Completion Rate | Percentage of treatments completed | > 90% |
| Risk Acceptance Rate | Percentage of risks accepted | < 10% |

### Setting Targets
1. Go to **Settings → Risk Configuration**
2. Set target thresholds
3. Dashboard shows against targets

## Best Practices

### Daily Review
- Check top risks
- Review new risks

### Weekly Reporting
- Export dashboard summary
- Share with stakeholders

### Monthly Analysis
- Review trend data
- Identify patterns
