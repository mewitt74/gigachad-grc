# Trust Analytics

Trust Analytics provides comprehensive insights into your questionnaire management, response times, and trust team performance.

## Overview

Trust Analytics helps you:
- Monitor questionnaire completion rates and trends
- Track SLA compliance and response times
- Identify bottlenecks in the review process
- Measure team performance and workload
- Optimize your trust program

## Accessing Analytics

Navigate to **Trust → Trust Analytics**

### Dashboard View
The main analytics dashboard displays:
- Key metrics summary
- Trend charts
- SLA status
- Team performance overview

## Key Metrics

### Questionnaire Metrics

#### Completion Rate
Percentage of questionnaires completed on time:
- **Target**: 90%+ completion rate
- **Calculation**: (Completed on time / Total received) × 100

#### Average Response Time
Mean time to complete questionnaires:
- Measured from submission to completion
- Broken down by complexity
- Compared to SLA targets

#### Volume Trends
Questionnaire volume over time:
- Weekly/monthly/quarterly views
- Year-over-year comparison
- Seasonal patterns

### Question Metrics

#### Questions Processed
Total questions answered:
- By category
- By analyst
- By time period

#### Answer Sources
Where answers come from:
- **Knowledge Base**: Auto-suggested from KB
- **Templates**: Applied from templates
- **AI Drafted**: Generated by AI
- **Manual**: Written from scratch

#### Reuse Rate
How often existing answers are reused:
- High reuse = efficient process
- Low reuse = potential KB gaps

### SLA Metrics

#### On-Time Completion
Percentage meeting SLA deadlines:
- By priority level (urgent/high/normal/low)
- By requester type
- By questionnaire size

#### SLA Breach Analysis
When SLAs are missed:
- Root cause breakdown
- Breach frequency
- Impact assessment
