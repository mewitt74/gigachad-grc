# Report Builder

Create custom reports tailored to your specific needs with the Report Builder.

## Overview

The Report Builder allows you:
- Build custom reports visually
- Combine data from multiple sources
- Create reusable templates
- Export in multiple formats

## Accessing Report Builder

Navigate to **Tools â†’ Report Builder**

## Building Reports

### Create New Report
1. Click **New Report**
2. Enter:
   - **Name**: Report title
   - **Description**: Report purpose
3. Choose layout:
   - Single page
   - Multi-section
   - Dashboard style

### Add Sections

#### Header
- Report title
- Date generated
- Organization logo
- Custom text

#### Data Tables
Add data tables:
1. Click **Add Table**
2. Select data source:
   - Controls
   - Risks
   - Evidence
   - Vendors
   - Audits
3. Choose columns
4. Apply filters
5. Set sorting

#### Charts
Add visualizations:
1. Click **Add Chart**
2. Select chart type:
   - Bar chart
   - Pie chart
   - Line chart
   - Heatmap
3. Configure data source
4. Customize appearance

#### Summary Metrics
Add key numbers:
- Total counts
- Percentages
- Averages
- Custom calculations

#### Text Blocks
Add narrative:
- Static text
- Dynamic placeholders
- Formatted sections

### Configure Sections

#### Column Selection
Choose which fields to include:
- Drag to reorder
- Rename headers
- Set width

#### Filtering
Add conditions:
- Status equals
- Date range
- Category is
- Custom filters

#### Sorting
Order results:
- Primary sort
- Secondary sort
- Ascending/descending

### Layout Options

#### Page Setup
- Page size (Letter, A4)
- Orientation (Portrait, Landscape)
- Margins

#### Styling
- Colors and themes
- Font selection
- Branding elements

## Report Templates

### Save as Template
1. Build report
2. Click **Save as Template**
3. Name template
4. Use for future reports

### Use Template
1. Click **New Report**
2. Select **From Template**
3. Choose template
4. Customize as needed

### Share Templates
- Share with team
- Organization-wide templates
- Lock templates from editing

## Preview and Export

### Preview
1. Click **Preview**
2. Review report
3. Check formatting
4. Verify data

### Export Options
| Format | Best For |
|--------|----------|
| **PDF** | Sharing, printing |
| **Excel** | Data analysis |
| **CSV** | Raw data |
| **Word** | Editing |
| **PowerPoint** | Presentations |

### Generate Report
1. Click **Generate**
2. Select format
3. Download or email

## Advanced Features

- `{{today}}`
- `{{start_of_quarter}}`

- Yellow for warning

### Cross-References
Link data sources:
- Controls with evidence
- Risks with treatments
- Vendors with assessments

### Calculated Fields
Create formulas:
- Sum totals
- Calculate percentages
- Custom calculations
