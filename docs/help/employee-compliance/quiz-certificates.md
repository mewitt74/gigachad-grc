# Quiz Engine & Certificates

Assess learning and issue verifiable completion certificates.

## Quiz Engine Overview

The quiz engine provides:
- Multiple question types
- Configurable passing scores
- Randomization options
- Immediate feedback
- Retry policies

## Question Types

### Multiple Choice

Single correct answer from options:

```
Q: What is the safest way to handle a suspicious email?

A) Reply asking for more information
B) Forward to the IT security team ✓
C) Click the link to investigate
D) Delete it immediately
```

### True/False

Binary answer questions:

```
Q: It's safe to share passwords with IT support for troubleshooting.

True / False ✓
```

### Multi-Select

Multiple correct answers:

```
Q: Which are signs of a phishing email? (Select all that apply)

☑ Urgent language demanding immediate action
☑ Misspelled sender domain
☐ Company logo in signature
☑ Request for login credentials
```

### Matching

Connect related items:

```
Match the threat to its description:
- Phishing → Email-based credential theft
- Malware → Malicious software installation
- Ransomware → Data encryption for payment
```

## Taking a Quiz

### Starting the Quiz

1. Open assigned training course
2. Complete course content
3. Click **Take Quiz**
4. Read instructions
5. Click **Begin**

### During the Quiz

- Answer all questions
- Flag questions to review
- Track time (if timed)
- Submit when complete

### Quiz Results

After submission:
- See overall score
- Review correct answers
- View explanations
- Retake if allowed

## Quiz Configuration

### Passing Score

Set minimum score to pass:
- Default: 80%
- Configurable per course
- Determines certificate eligibility

### Question Randomization

Options for randomization:
- Shuffle question order
- Shuffle answer choices
- Random question subset

### Time Limits

Configure quiz duration:
- No limit (recommended)
- Timed (stress testing)
- Per-question limits

### Retry Policies

Control retake attempts:
- Unlimited retries
- Limited attempts (e.g., 3)
- Waiting period between attempts

## Certificates

### Automatic Generation

Certificates generate when:
- Quiz score meets minimum
- All course content completed
- Prerequisites satisfied

### Certificate Contents

Each certificate includes:
- Employee name
- Course title
- Completion date
- Unique certificate ID
- Organization name
- Verification QR code

### Downloading Certificates

1. Go to **My Training → Completed**
2. Click on completed course
3. Click **Download Certificate**
4. PDF downloads automatically

### Certificate Verification

Verify certificate authenticity:

1. Scan QR code on certificate
2. Or visit verification URL
3. Enter certificate ID
4. View verification status

## Administrator Features

### Quiz Management

#### Creating Questions

1. Go to **Training → Courses → Quiz**
2. Click **Add Question**
3. Select question type
4. Enter question and answers
5. Mark correct answer(s)
6. Add explanation (optional)
7. Save

#### Question Bank

Maintain a question bank:
- Reuse across courses
- Categorize by topic
- Track question performance
- Update based on feedback

#### Quiz Analytics

View quiz performance:
- Average scores
- Question difficulty
- Pass/fail rates
- Time to complete

### Certificate Management

#### Certificate Templates

Customize certificate appearance:
- Logo placement
- Color scheme
- Layout options
- Signature images

#### Bulk Certificate Generation

Generate certificates in bulk:
- Select completed users
- Generate all at once
- Email to users
- Download as ZIP

#### Certificate Revocation

Revoke certificates if needed:
- For policy violations
- Termination
- Training invalidation
- Maintains audit trail

- Make certificates meaningful
- Display achievement
- Link to career development
- Recognize completion

### Continuous Learning

- Update content regularly
- Refresh annually

## Troubleshooting

### Quiz Won't Submit

- Check network connection
- Ensure all questions answered
- Try different browser
- Contact administrator

### Certificate Not Generated

- Verify passing score achieved
- Check all content completed
- Wait for processing
- Contact administrator

### Can't Verify Certificate

- Check certificate ID
- Ensure QR code readable
- Use correct verification URL
- Certificate may be revoked
