## Page Layouts

### List Pages
Most list pages (Controls, Risks, Vendors) have:
- **Search Bar**: Filter by text
- **Filters**: Category, status, date range
- **Sort**: Column headers are clickable
- **Actions**: Add, Export, Bulk Upload
- **Pagination**: Navigate large lists

### Detail Pages
Detail pages for individual items show:
- **Header**: Title, ID, status badge
- **Actions**: Edit, Delete, more options
- **Tabs**: Different sections of information
- **Related Items**: Linked entities
- **Activity**: History and changes

### Settings Pages
Settings are organized by category:
- **Cards**: Each setting group in a card
- **Forms**: Input fields with descriptions
- **Save Buttons**: Per-section saving

## Tips

1. **Collapse Sidebar**: Click the hamburger icon to maximize content area
2. **Right-Click Context Menus**: Many items have context menus
3. **Breadcrumbs**: Use breadcrumbs at the top to go back
4. **Recent Items**: Dashboard shows your recently viewed items
