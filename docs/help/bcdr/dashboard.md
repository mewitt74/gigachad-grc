# BC/DR Dashboard

The Business Continuity and Disaster Recovery (BC/DR) Dashboard provides an overview of your organization's resilience posture.

## Overview

The dashboard displays:
- Critical business process status
- Recovery plan readiness
- DR test results and schedule
- Key resilience metrics

## Key Metrics

### Business Impact Analysis
- Critical processes identified
- Impact assessments completed
- RTO/RPO defined

### Plan Coverage
- Processes with recovery plans
- Plan review status
- Gap analysis

### DR Testing
- Tests completed this year
- Test success rate
- Next scheduled test

## Dashboard Widgets

### Process Criticality
Breakdown of processes by criticality:
- Critical (must recover immediately)
- High (recover within hours)
- Medium (recover within days)
- Low (recover within weeks)

### Recovery Readiness
Status of recovery preparations:
- Documented plans
- Tested procedures
- Current vs. target

### Test Calendar
Upcoming and recent DR tests:
- Test dates
- Scope/scenario
- Results summary

### Incident Timeline
Recent BC/DR-related incidents:
- Invocations
- Near misses
- Lessons learned
